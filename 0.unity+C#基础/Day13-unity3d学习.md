## U3D

### 杂记

- 如果要生成对象，则需要确保使用存储在“项目”窗口中的预制件（从项目窗口拖出来）（而不是层级视图里的！）
- Random.Range(-20, 20)
- Start() 中 ，使用 InvokeRepeating 根据间隔生成，methodname 大小写敏感
- OnTriggerEnter(Collider other)处理碰撞
- 字体导入后右键 create font asset
- 碰撞是 box collider
